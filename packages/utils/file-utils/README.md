# @heroui/file-utils

Utilidad interna para el manejo de los archivos

Please refer to the [documentation](https://heroui.com/docs/components/file-utils) for more information.

## Installation

```sh
yarn add @heroui/file-utils
# or
npm i @heroui/file-utils
```
